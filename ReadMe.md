# MultiCam Manager

Der MultiCam Manager ist eine Software um mehrere Kameras simultan zu steuern und überwachen zu können.  

## Inhalt
1. Installation  

## 1. Installation
### 1.1 Requirements
Folgene Software ist zu installieren um den MultiCam Manager zu benutzen:  
* Qt 6.5 LTS 
* CMake
* OpenCV 4.12
* MSVC 2022 C++ Compiler 

#### Qt 6.5 LTS 
Kann [hier](https://doc.qt.io/qt-6.5/windows.html) heruntergeladen und über das Qt Maintenance Tool installiert werden.

#### CMake
Kann [hier](https://cmake.org/download/) herunterladen werden.

#### OpenCV 4.12
Kann [hier](https://opencv.org/releases/) heruntergalden werden.

#### MSVC 2022 C++ Compiler
Im Visual Studio installer für VS Studio 2022 müssen die Komponenten für C++ Desktop Entwicklung installiert sein.  
Um zu debuggen muss das Windows 10/11 SDK Kit [hier](https://developer.microsoft.com/en-us/windows/downloads/windows-sdk/) heruntergeladen werden.

### 1.2 Initialisierung 
Sind alle Requirements richtig installiert dann kann der MultiCam Manager [hier](https://gitlab.rz.htw-berlin.de/softwareentwicklungsprojekt/wise2025-26/team-01-multikamera-manager-gfai-bvvi) über das HTW GitLab bezogen werden.  
Bei Butzung der **Qt IDE** die CMakeLists.txt öffnen, das Projekt bauen und starten.
